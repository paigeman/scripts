# scripts

### 仓库的内容

一些个人在日常生活中使用的脚本

### 脚本的类型

- `Debian` 的 `apt-sources.list`
- `docker` 的安装脚本
- 一些 `Dockerfile`
- `Linux` 下的一些 `desktop` 文件
- `kubernetes` 及其衍生产品安装部署用到的一些列脚本
- `maven` 的配置文件（国内使用）
- `Clash for Windows` 在 `Linux` 下的安装脚本
- `nftables` 防火墙的配置文件
- 恢复Windows默认应用的注册表（ `cmd` 、`batch` ...）
- 其它一些脚本
